# Bulk Rename Incrementing Prefix

Want to move over to Obsidian? But you do not want to lose the order the notes were stored in (last modified?) as obsidian stores in alphabetical order?



Then use this simple plugin!



FirstNote

SecondNote

SomeNote

LastNote



Turns into:



001 FirstNote

002 SecondNote

003 SomeNote

004 LastNote


Works like a charm!

\-----------------------------

# How to use it?

It's quite simple. Just go to Tools > Options > Plugins > Gearwheel Icon > Install from file > Upload the .JPL file.



The others are just for the sake of open source the tech savier ones know how to work with it. Vibecoded in a minute!
